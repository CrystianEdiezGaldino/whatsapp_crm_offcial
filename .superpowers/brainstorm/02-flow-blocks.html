Blocos de Fluxo Conversacional 🧩

Componentes que o admin pode usar para montar o fluxo

Blocos Principais
💬
Enviar Mensagem
Envia uma mensagem automática pro cliente. Pode ter texto, emojis, e variáveis.
Ex: "Olá {nome}, bem-vindo! 👋"
🎯
Menu / Decisão
Apresenta opções numeradas para o cliente escolher. Roteia para fluxos diferentes.
Ex: "Digite 1 para Suporte, 2 para Admin"
⚙️
Ação / Sistema
Executa uma ação: transfere para agente, marca tags, registra informação.
Ex: "Transferir para Suporte", "Adicionar tag: premium"
⏱️
Aguardar / Delay
Pausa antes da próxima mensagem. Simula conversa natural.
Ex: Aguarda 2 segundos
👤
Transferir para Agente
Entrega o fluxo para um agente real atender. Pode transferir para pessoa específica ou fila.
Ex: "Transferir para Suporte Técnico"
❓
Condicional
Toma decisão baseada em dados (resposta do cliente, hora do dia, etc).
Ex: "Se hora > 18h, enviar mensagem diferente"

Exemplo: Fluxo de Novo Cliente 📊

1
💬 Enviar Mensagem (Boas-vindas)
"Olá! Bem-vindo ao SisChat. Como posso ajudar?"
2
⏱️ Aguardar 1 segundo
Pausa natural antes do menu
3
🎯 Menu com opções
"Digite: 1=Dúvida, 2=Problema, 3=Outro"
4
💬 Mensagem (Confirmação)
Se escolheu opção 1: "Ótimo! Estamos analisando..."
5
👤 Transferir para Agente
Passa para fila de Suporte com contexto

💡 Nota: Os blocos acima são os principais. Conforme evolui, pode adicionar mais: WebHook, delay condicional, etc.
