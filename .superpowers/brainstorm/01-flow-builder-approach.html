Sistema de Fluxo Conversacional 🤖

Qual abordagem o admin prefere para criar fluxos?

🔗 Node-Based Builder
Visual Canvas com Nodes
Arrasta nodes (Message, Decision, Action) em um canvas. Conecta com linhas. Tipo Zapier/Make.

✓ Vantagens

  • Visual intuitivo
  • Fluxos complexos fáceis
  • Ramificações claras

✗ Desvantagens

  • Mais complexo de dev
  • Precisa de JS robusto
  • Parsing de grafo
📋 Form-Based (Step by Step)
Wizard com Formulários
Campos em abas/steps. "Mensagem inicial → Opção 1 → Ação". Tipo Wix/Webflow.

✓ Vantagens

  • Simples de dev
  • Mais focado
  • Menos bugs

✗ Desvantagens

  • Fluxos complexos chatos
  • Difícil visualizar
  • Limite de níveis
⚡ Hybrid (Recomendado)
Visual Light + Form Config
Canvas simples (drag+drop básico) + painel de config. Inicio em node-based, evolui para form.

✓ Vantagens

  • Bom custo-benefício
  • Escalável
  • Intuitivo + robusto

✗ Desvantagens

  • Meio do caminho
  • Requer design bom
Nenhuma opção selecionada
