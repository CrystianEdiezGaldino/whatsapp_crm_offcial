Feedback System XSS Security Tests

These tests verify that HTML/JavaScript injections in titles, messages, and callbacks are properly escaped and never executed.

Test 1: HTML Injection in confirm() title

This test passes an <img> tag with onerror handler in the title parameter.

Test 2: HTML Injection in confirm() message

This test passes a <script> tag in the message parameter.

Test 3: HTML Injection in alert() title

This test passes an <img> tag with onerror handler in the alert title.

Test 4: HTML Injection in alert() message

This test passes a <script> tag in the alert message.

Test 5: HTML Injection in toast() message

This test passes an <img> tag with onerror handler in the toast message.

Test 6: Callback functions execute safely

This test verifies that callbacks are executed properly when buttons are clicked.

Test 7: DOM Inspection - No innerHTML from user input

This test inspects the DOM to verify no user input was set via innerHTML.
